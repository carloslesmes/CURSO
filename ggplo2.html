1 Paquete ggplot2

El paquete ggplot2 es un paquete que hace gráficas a partir de los datos. Primero instalarlo con install.packages("ggplot2") y activarlo con library(ggplot2). Los paquetes ggExtra y ggthemes son paquetes adicionales que ayudan a mejorar sus gráficas, de igual forma se instalan y se activan.

1.1 Usos de ggplot2

Activamos los paquetes y cargamos el archivo de datos mpg del mismo. Haga data(package="ggplot2") en la consola de R, para ver los archivos de datos del paquete. Usaremos el archivo de datos mpg. El paquete ggplot2 trabaja únicamente a partir de un dataframe o tibble.

library(ggplot2)
library(ggExtra)
library(ggthemes)

2 Pasos para hacer una gráfica con ggplot2

2.1 Primer paso

Escribir ggplot(data = <dataframe>) o ggplot(<dataframe>)

Código 
ggplot(data=mpg)

Al correr esta instrucción solo se obtiene el plano donde se irán agregando capas.

2.2 Segundo paso

Agregamos una geometría con la(s) variable(s) que se van a graficar.

Código 
ggplot(data=mpg) +
geom_point(mapping = aes(x=displ, y=hwy))
Código 
ggplot(mpg) +
geom_point(mapping = aes(x=displ, y=hwy, color=class))
Código 
ggplot(mpg) +
geom_point(aes(x=displ, y=hwy, color=class))

2.3 Tercer paso

Hacer facetas:

Código 
ggplot(mpg) +
geom_point(mapping = aes(x=displ, y=hwy, color=class)) +
facet_wrap(~ class)
Código 
ggplot(mpg) +
geom_point(mapping = aes(x=displ, y=hwy, color=class)) +
facet_grid(drv ~ cyl)

9 Presentación de varios gráficos

Código 
# Data simulation
set.seed(5)
x <- runif(100)
df <- data.frame(x = seq_along(x), var = x)

# Box plot
p1 <- ggplot(df, aes(x = "", y = var)) +
  geom_boxplot(fill='yellow')

# Density plot
p2 <- ggplot() +
  stat_function(fun = dnorm, geom = "density",
                xlim = c(-3, 3), fill = "lightskyblue") 

# Line chart
p3 <- ggplot(df, aes(x = x, y = var)) +
  geom_line(color = "pink")

# Scatter plot
p4 <- ggplot(df, aes(x = x, y = var)) +
  geom_point(color = "green")

# View the plots
